Flex Properties are applied to flex items, which are the children of a flex container.

flex-grow determines how much a flex item will grow relative to the rest of the flex items.
flex-shrink determines how much a flex item will shrink relative to the rest of the flex items.
flex-basis sets the initial main size of a flex item before the remaining space is distributed.

Flex grow

This property defines the ability of a flex item to grow if necessary. It's a numerical value that represents the proportion of available space in the flex container that the item should take up.

align-items: stretch (on the container): This property will stretch all child elements to fit the container's cross-axis (vertical axis if the flex direction is row, horizontal axis if the flex direction is column), assuming the children do not have a cross-size (height or width depending on the flex direction) set. It doesn't affect the main-axis size (horizontal size for a row, vertical size for a column).
flex-grow (on the child): This property allows a child element to grow to absorb any extra space along the main-axis of the container. If flex-grow is set to a positive value, the flex item will grow to take up available space on the main-axis, in proportion to other flex items' flex-grow values. So, align-items: stretch ensures that flex items fill the container along the cross-axis, while flex-grow allows flex items to expand and fill the remaining space on the main-axis. Both can be used together to make flex items fill the entire space of the flex container in both dimensions.

Flex Shrink

This property defines the ability of a flex item to shrink if necessary. Like flex-grow, it's a numerical value that represents how much the flex item will shrink relative to the rest of the flex items in the container when there isn't enough space. A higher flex-shrink value means the item will shrink more than items with a lower flex-shrink value.
