Justify-content(main axis - flex-start, flex-end, center, space-between, space-around, space-evenly) and align-items (cross axis - flex-start, flex-end, center, baseline, stretch) are used for parent container

Align-self is used in child

align-content (with flex-wrap enabled): This property aligns the flex container's lines within the flex container when there is extra space in the cross-axis, similar to how justify-content aligns individual items within the main axis.
